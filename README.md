# Job Hunter Skills

求职skills

## 项目结构

```
job-hunter-skills/
├── job-hunter/          # 🧭 求职总入口，智能路由到子技能
├── job-fit-analyzer/    # 📊 岗位匹配度分析，五维评分 + HTML 报告
├── interview-coach/     # 🎯 面试教练，个性化问题 + STAR 答案
├── docx/                # 📝 Word 文档生成，简历/报告输出
└── LICENSE              # Apache 2.0
```

## 技能说明

### 🧭 job-hunter — 求职总入口

智能识别用户意图，自动路由到对应子技能：

- **模式 A：新求职** — 结构化收集求职偏好（目标职位、城市、薪资、公司规模等），建立求职画像
- **模式 B：岗位匹配** — 用户已有简历 + JD，自动跳转 `job-fit-analyzer` 进行匹配度分析
- **模式 C：面试准备** — 用户即将面试，自动跳转 `interview-coach` 生成面试辅导
- **模式 D：简历优化** — 针对目标岗位优化现有简历

### 📊 job-fit-analyzer — 岗位匹配度分析

将职位描述与候选人简历进行五维度量化对比：

| 维度 | 权重 | 说明 |
|------|------|------|
| 硬技能深度与精准匹配度 | 28% | 核心技术栈掌握深度和实际生产经验 |
| 过往成果与业务影响力 | 25% | 可量化的业务成果和个人贡献占比 |
| 职责范围、复杂度与 Seniority 匹配度 | 20% | 项目规模、独立负责程度、决策层级 |
| 领域/行业/场景经验相关性 | 15% | 业务领域、行业特性、场景匹配度 |
| 学习能力、潜力和适应性 | 12% | 快速上手新技术/新领域的记录和潜力 |

输出：一份自包含的 HTML 报告，包含综合评分、各维度得分、优势与短板、竞争力评估、可执行建议。

### 🎯 interview-coach — 面试教练

基于简历和 JD 的交叉分析，提供个性化面试准备：

- **角色技术题**（5-10 题）— 针对目标岗位的专业技术问题
- **行为面试题**（8-12 题）— 基于 STAR 框架的结构化回答
- **简历深挖题**（5-8 题）— 基于简历经历细节的追问
- **缺口防御题**（3-5 题）— 针对候选人短板的预设回应
- **追问预测** — 每个问题附带面试官可能的连环追问

输出：交互式 HTML 面试准备页面。

### 📝 docx — Word 文档工具

底层文档生成能力，支持：

- 创建格式化的 .docx 文档（简历、报告、信函）
- 读取和解析现有 Word 文档
- 编辑已有文档内容
- 图片插入与替换
- 目录、页眉页脚、页码等高级格式

## 典型工作流

```
用户说"帮我找工作"
    ↓
job-hunter 收集偏好 → 建立求职画像
    ↓
用户提供 JD → job-fit-analyzer 五维分析 → HTML 报告
    ↓
匹配度 ≥ 65 → interview-coach 面试辅导 → 面试准备页
    ↓
docx 输出优化后简历
```
